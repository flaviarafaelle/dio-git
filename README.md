# Comandos básicos
## Windowns
* dir: mostra uma lista de diretórios
* cd: navegar entre pastas
* cd..: retroceder
* cls: limpar
* TAB: completar
* mkdir nomedapasta: criar pasta
* echo frase > frase.txt: cria e printa uma frase/termo
* del/rmdir: deletar
